Instalar AutoBam

AutoBam es una herramienta ligera diseñada para mejorar tu flujo de trabajo con Bambu Studio. Puedes instalar la aplicación haciendo clic en el siguiente botón:

Instalar AutoBam

La instalación se actualizará automáticamente cuando haya nuevas versiones.

Requisitos previos

Necesitas tener instalado:

  • Microsoft .NET Framework 4.8.1

Si lo prefieres, también puedes descargar el instalador completo:

Descargar instalador
